gundi

Unearth your forgotten code debt — TODO scanner with git blame age tracking

cargo install gundi click to copy
Star on GitHub

What it does

🔍

Scan everything

Finds TODO, FIXME, HACK, BUG, and XXX comments across your entire codebase with parallel file walking.

📅

Age tracking

Runs git blame on each comment to show who wrote it, when, and how many days it has been sitting there.

🚨

CI gate

Use --fail-on 90 to block merges when any debt is older than your threshold.

🎯

Filter and export

Filter by type, author, or age. Output as a colored table, JSON, or Markdown for reports and dashboards.

See it in action

terminal
$ gundi .
TYPE FILE                                 LINE   AUTHOR               AGE          TEXT
--------------------------------------------------------------------------------------------------------------
TODO   src/parser.rs                          42     alice                142d ago      refactor this loop
FIXME  src/handler.rs                         87     bob                  63d ago       handle timeout case
HACK   tests/integration.rs                   15     charlie              8d ago        temporary workaround
TODO   src/config.rs                          201    alice                310d ago      support YAML format
Total: 4 items

Red = older than 90 days · Yellow = older than 30 days · Green = recent
